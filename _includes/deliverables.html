Final Team Deliverables

View required deliverables

Each team must complete and submit the following deliverables before their final presentation on the last day of the hackathon.

GitHub Repository

Each team must maintain a GitHub repository that includes:

  • README.md with the project description and goals
  • Source code for the project
  • Comments in the code explaining key functionality
  • Any files needed to run, view, or understand the project

Project Poster

Each team must submit a project poster in PDF format.

Final Presentation

Each team must submit a final presentation in PDF format. The presentation should include:

  • Team members’ names
  • Team members’ schools
  • Team members’ photos
  • Technology used, such as tools, HPC resources, AI resources, libraries, datasets, or platforms
  • Chosen Innovation Track
  • Innovation Track Visualization of the chosen theme

Website, Visualization, or Application

Each team must provide a website, visualization, or application that demonstrates their project idea and findings.

Proof of Poster Submission

Each team should provide evidence that their project poster was submitted to Gateways 2026.

Note: Teams that submit proof of poster submission to Gateways 2026 will receive +5 points added to their judging score.
